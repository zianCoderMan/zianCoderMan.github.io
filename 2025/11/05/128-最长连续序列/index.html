128. 最长连续序列

zianCoderMan
2025-11-05
算法刷题
LeetCode Top100, 哈希

问题与示例：

给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 O(n) 的算法解决此问题。

题解：

结合例子 num_set = {100,4,200,1,3,2} 详细走一遍：

  1. 遍历到 100：
    • 判断 100-1=99 是否在集合中？否 → 是起始元素；
    • currentNum=100，currentStreak=1；
    • 检查 100+1=101 是否在集合中？否 → 循环结束；
    • longestStreak = max(0,1) = 1。
  2. 遍历到 4：
    • 判断 4-1=3 是否在集合中？是 → 不是起始元素，跳过。
  3. 遍历到 200：
    • 判断 200-1=199 是否在集合中？否 → 是起始元素；
    • currentNum=200，currentStreak=1；
    • 检查 201 不在集合 → 循环结束；
    • longestStreak = max(1,1) = 1。
  4. 遍历到 1：
    • 判断 1-1=0 是否在集合中？否 → 是起始元素；
    • currentNum=1，currentStreak=1；
    • 检查 1+1=2 在集合中 → currentNum=2，currentStreak=2；
    • 检查 2+1=3 在集合中 → currentNum=3，currentStreak=3；
    • 检查 3+1=4 在集合中 → currentNum=4，currentStreak=4；
    • 检查 4+1=5 不在集合 → 循环结束；
    • longestStreak = max(1,4) = 4。
  5. 遍历到 3：
    • 判断 3-1=2 在集合中 → 不是起始元素，跳过。
  6. 遍历到 2：
    • 判断 2-1=1 在集合中 → 不是起始元素，跳过。

最终 longestStreak=4，正确。
